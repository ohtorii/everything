秀丸エディタからEverythingを使用する秀丸マクロ
========


###動作確認を行った環境
- 秀丸エディタ v8.40 beta10 32bit
- Everything-1.3.3.658b.x64.exe
- Windows7 64bit

###カスタマイズ可能な項目
* 検索結果の表示方法
 * アウトプット枠
 * 新規ファイルへ出力
 * カーソル位置へ挿入

* 検索文字列の指定
 * 入力ボックス
 * カーソルのある行の文字
